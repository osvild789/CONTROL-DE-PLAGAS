Simulador de Plagas — Fibonacci & Números Primos
Proyecto del Desafío Web 2025: Matemáticas que resuelven problemas.
Descripción
Página web interactiva que simula el crecimiento de una población de insectos plaga
usando la Serie de Fibonacci. Detecta etapas críticas con números primos.
Estructura
desafio-fibonacci-primos/
├── index.html
├── css/
│   └── estilos.css
├── js/
│   └── script.js
└── README.md
Cómo usar

Abre index.html en tu navegador.
Ingresa el número de meses, el tipo de insecto y el umbral de alerta.
Presiona Simular crecimiento.

Tecnologías

HTML5 — estructura semántica
CSS3 — diseño responsivo con variables CSS
JavaScript puro — Fibonacci sin arrays, verificación de primos
