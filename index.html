01 — Contexto del problema

¿Por qué las plagas crecen como Fibonacci?

🌿

El problema real

Los agricultores y biólogos necesitan predecir cuántos insectos plaga habrá en los próximos meses para decidir cuándo y cuánto pesticida aplicar. Una predicción errónea puede arruinar una cosecha o desperdiciar recursos.

📐

La solución matemática

Muchas plagas (pulgones, saltamontes, langostas) tienen ciclos reproductivos que siguen patrones similares a Fibonacci: cada generación depende de las dos anteriores. Esto nos permite proyectar la infestación con sorprendente precisión.

⚠️

Cuándo actuar

El simulador detecta automáticamente si algún término de la serie es también número primo, lo que indica una etapa crítica donde la plaga es especialmente vulnerable a ser controlada.

02 — Algoritmo

Cómo funciona la simulación

F(1)
Empezamos con 1 insecto en el mes 1 (población inicial mínima).
F(2)
En el mes 2 hay 1 insecto (la primera reproducción no produce nuevos individuos aún).
F(n)
A partir del mes 3: F(n) = F(n-1) + F(n-2). Cada mes, la nueva población es la suma de los dos meses anteriores.
🔴
Si el número de insectos en un mes es primo, se marca como etapa crítica: momento ideal para intervención.
// Fibonacci sin arrays (solo variables simples)
let a = 0, b = 1, c;
for (let i = 2; i <= meses; i++) {
  c = a + b;   // nueva población = suma de las dos anteriores
  a = b;
  b = c;
}

05 — Conclusión

¿Cómo ayuda la matemática?

La Serie de Fibonacci no es solo un ejercicio abstracto: modela fenómenos reales de crecimiento en la naturaleza. Al aplicarla al crecimiento de plagas, los agricultores pueden anticiparse a infestaciones masivas antes de que ocurran, reduciendo costos y daños.

La combinación con números primos agrega una capa de análisis: identificar etapas "únicas" en el ciclo de vida de la plaga permite planificar el momento más efectivo para la intervención, optimizando el uso de pesticidas y protegiendo el medio ambiente.

Este proyecto demuestra que las matemáticas son una herramienta práctica para resolver problemas del mundo real, desde la agricultura hasta la seguridad informática.
